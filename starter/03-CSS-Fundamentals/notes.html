Ch 24. Styling Text

Serif means those short, spike-like edges of text.

The style of an element affects the style of its child elements.

The style 'text-align: center' positions the text at the center of parent element.

The default font-size is 16px.

Ch. 25 Combining Selectors

A list of selectors can be supplied for a CSS rule (known as a 'list-selector').

A string of selectors, separated by white-space, can be supplied for a CSS rule (known as a 'descendent-selector').

Descendent-Selectors

Descendent-selectors are incredibly useful, but if it is written as to reflect the HTML structure, it will lead to code that is harder to maintain in the future.
